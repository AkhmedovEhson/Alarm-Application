# Alarm-Application
Welcome to Alarm API ! 🌐 
There are bunch of API routes. Just schedule, Alarm will send email in time 📧

# Versions
* .NET 7 ⚡️
* React 18.0 🎨

# Security
* #️⃣ Uses BCrypt, user's password is stored in `SHA256` hash type.
* 🔒 Provide JWT authorization.
